# Personal Expense Tracker (Excel)

## Overview
This project is an Excel-based dashboard designed to track and analyze personal expenses efficiently.

## Features
- Category-wise expense analysis
- Total expense calculation
- Expense distribution using charts (Pie/Bar)
- Insights on spending patterns

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Purpose
To help users understand their spending habits and make better financial decisions.
